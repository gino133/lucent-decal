# 🪟 Lucent Glass – Website Fullstack + CMS

Hệ thống quản trị nội dung website cho thương hiệu **Lucent Glass** – chuyên về decal kiến trúc và kính nghệ thuật. Cho phép quản trị viên chỉnh sửa toàn bộ nội dung, menu, sản phẩm, dự án, và giao diện như WordPress.

---

## 🚀 Công nghệ sử dụng

### Frontend
- React 18
- React Router DOM 6
- Tailwind CSS
- Axios
- React Hook Form

### Backend
- Node.js + Express
- MongoDB (Mongoose)
- JWT Authentication
- Bcryptjs

### Deploy
- **Frontend**: Vercel
- **Backend**: Render
- **Database**: MongoDB Atlas
